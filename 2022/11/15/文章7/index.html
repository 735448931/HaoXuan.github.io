jQuery学习

发表于2022-11-15|更新于2022-11-15|jQuery

jQuery 第一天

1.选择器

jQuery 中通过选择器来获取 DOM 节点，功能类似于原生的 querySelectorAll 方法，支持的选择器与 CSS 的选择器几乎一致。

2.对象

jQuery中利用选择器获取到的并非原生的DOM对象，而是jQuery对象

// 选择器获取
$(‘选择器’)
// dom对象转换
$(dom对象)

3.事件绑定

在jQuery中以原生事件类型的名称为依据，封装了想对应的事件处理方法

  • 事件名开头不需要写on
  • 回调函数中的this就是触发事件的dom元素

4.链式编程

链式编程通过点.把多个操作(方法)连续的写下去，形成和链子一样的结构

原：

现：$(‘.text’).focus(回调函数).blur(回调函数).change(回调函数)

大部分jQ对象方法的返回值还是同一个jQ对象

5.内容操纵

jQuery中封装了设置和读取网页元素文本内容的方法

设置时： html方法解析标签，text不解析标签
取值时：html方法获取标签，text只获取文本

只有设置支持链式编程

6.过滤方法

jQuery中封装了过滤方法，对jQuery对象中的dom元素再次筛选

  • eq方法阿德索引从0开始
  • 返回的是jQuery对象

7.样式操纵

jQuery中对样式的操作进行封装，可以设置或者获取样式

数值类的样式省略单位，默认会使用px
获取样式需要传递样式名

8.属性操纵

jQuery中对属性的操作进行封装，可以设置，获取和删除属性

9.操纵value

jQuery中封装了操纵表单元素value属性的方法，可以取值和赋值

10.查找方法

11.操纵类名

jQuery中封装了为网页元素添加，移除，检测，切换类名的方法

12.事件进阶

jQuery中封装了更为灵活的 on/off，one方法处理DOM事件

13.触发事件

jQuery中通过代码的方式触发绑定的事件

14.window事件绑定

使用jQuery为window对象绑定事件

jQuery 第二天

1.获取位置

通过jQuery直接获取元素的位置

区别：

​ 1.offset 参照 html 标签

​ 2.position 参照离他最近有定位的祖先元素

​ 3.offset 会把外边距margin计算进去

​ 4.position以外边距margin为边界，不计算margin

2.滚动距离

注册页面的滚动事件使用window.scroll()

获取页面的滚动距离使用 html标签作为选择器

设置滚动距离时没有动画效果

3.显示隐藏动画

通过jQuery以动画的方式切换元素的显示和隐藏

4.淡入淡出动画

通过jQuery以淡入淡出的方式切换元素的显示隐藏

淡入淡出效果是通过修改元素的 opactity 来实现的

5.展开收起动画

通过jQuery以展开和收起的方式切换元素的显示隐藏

6.动画队列及停止方法

通过jQuery为元素设置的多个动画会一次添加到动画队列中，并根据添加的顺序依次播放

动画方法和stop方法返回的是同一个jQuery对象（链式编程）

7.自定义动画

jQuery提供了animate方法来实现更为复杂的动画效果

8.插入节点

jQuery中封装了在指定位置动态插入元素节点的方法，可以插入节点或者改变节点位置

插入节点：传入创建的dom元素或者html结构
改变位置：传入现有的dom元素或者jQuery对象

9.动画的回调函数

所有的jQuery动画方法都支持传入回调函数

回调函数会在动画执行完毕时立刻执行

回调函数中的this是执行动画的dom元素

10.动画的延迟方法

jQuery不仅可以设置动画的执行的速度，还能在动画执行前设置一定的延时

11.获取尺寸

jQuery对获取元素尺寸进行了封装，使得在不同场景中获取元素尺寸十分方便

12.事件参数

jQuery绑定的时间中可以获取事件参数(事件对象)，用法和原生JS完全一致

13.删除节点

语法：jQuery对象.remove()

remove方法删除的是调用方法的元素节点

14.事件委托

jQuery中封装了事件委托的支持，直接通过on方法即可使用

减少事件注册
解决动态增加后代元素的事件绑定问题
原理是事件冒牌
回调函数中的this是触发事件的dom对象
